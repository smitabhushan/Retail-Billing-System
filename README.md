# 🛒 Retail Billing Software (POS System)

## 🚀 Project Overview
A production-ready full-stack Retail Billing / POS application built using **React.js (Vite)** and **Spring Boot**.

This project simulates a real-world billing system used in retail stores, covering product management, billing, payments, and analytics.

## 🛠️ Tech Stack

### 🔹 Frontend
- React.js (Vite)
- Bootstrap 5
- React Router DOM
- Axios

### 🔹 Backend
- Java
- Spring Boot
- Spring Security
- REST APIs

### 🔹 Database
- MySQL / MongoDB

### 🔹 Payment
- Razorpay

---

## 🔄 Application Workflow

- User/Admin logs in securely  
- Admin creates categories & items  
- Product images are uploaded  
- Items are added to cart  
- Customer details are entered  
- Payment is completed via Razorpay  
- Order is generated and stored  
- Dashboard displays sales & order data
