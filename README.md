# 🛒 Retail Billing Software (POS System)

## 🚀 Project Overview
A production-ready full-stack Retail Billing / POS application built using **React.js (Vite)** and **Spring Boot**.

This project simulates a real-world billing system used in retail stores, covering product management, billing, payments, and analytics.

## 🛠️ Tech Stack

### 🔹 Frontend
- React.js (Vite)
- Bootstrap 5
- React Router DOM
- Axios

### 🔹 Backend
- Java
- Spring Boot
- Spring Security
- REST APIs

### 🔹 Database
- MySQL / MongoDB

### 🔹 Payment
- Razorpay

---

## ✨ Features
- Category Management (CRUD)
- Product Management
- Cart & Billing System
- Customer Details
- Secure Authentication
- Role-Based Access (Admin/User)
- Search & Filter
- Order History
- Dashboard Analytics
- Image Upload (Local / AWS S3)
- Razorpay Integration
- Responsive UI
